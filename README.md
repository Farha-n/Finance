# Finance Dashboard

A finance dashboard to track transactions and analyze spending patterns.

## Tech Stack

- React (Vite)
- Tailwind CSS
- Recharts
- Context API

## Features

- Dashboard overview with summary cards and charts
- Transactions table with search, filter, and sorting
- Role-based UI (`viewer` / `admin`)
- Insights section for spending behavior
- Financial Health Score
- Local storage persistence

## Decisions

- Context API was used for simplicity and scalability in a single-page dashboard.
- Recharts was selected for composable chart components with quick readability.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
